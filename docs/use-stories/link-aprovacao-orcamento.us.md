### Modulo Orçamentos - Link de aprovação de orçamento

Como usuário, eu quero poder criar links de aprovação de orçamentos para enviar aos meus clientes, de forma que eles possam aprovar orçamentos diretamente pelo link, facilitando o processo de aprovação e agilizando o fechamento de negócios.

O link deve ser único para cada orçamento e permitir que o cliente visualize os detalhes do orçamento, incluindo itens, valores e condições, antes de aprová-lo.
